<task type="auto">
  <name>Task 2: Create job tracking tables</name>
  <files>lib/db/schema.ts</files>
  <action>
Add two job-related tables:

1. `refreshJobs` table:
- id: uuid primary key
- userId: text not null
- jobType: text not null (e.g., 'full-refresh', 'ranks-only')
- status: text not null ('queued', 'processing', 'complete', 'failed', 'cancelled')
- progress: integer (0-100)
- metadata: jsonb (stores job-specific data)
- startedAt: timestamp
- completedAt: timestamp
- errorMessage: text
- estimatedCost: real
- indexes: userId + status, startedAt

2. `jobHistory` table:
- id: uuid primary key
- userId: text not null
- jobId: uuid references refreshJobs
- eventType: text not null ('started', 'progress', 'completed', 'failed')
- eventData: jsonb
- createdAt: timestamp
- index: jobId, userId + createdAt

Extend type exports for RefreshJob, NewRefreshJob, JobHistory, NewJobHistory.
  </action>
  <verify>npx tsc --noEmit && grep -A 20 "refreshJobs" lib/db/schema.ts</verify>
