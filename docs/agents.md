# Docs

This directory contains specifications, guides, architecture docs, and improvement plans.

For project architecture, conventions, and agent patterns, see:
- [AGENTS.md](../AGENTS.md) — Main project knowledge base

## Key directories

- `docs/specs/` — Canonical product + architecture specifications
- `docs/guides/` — How-to guides and setup instructions
- `docs/plans/` — Improvement plans and milestones (gitignored)
- `docs/reports/` — Research reports and audits

## Note

`docs/plans/` is gitignored; commit durable docs under `docs/specs/` instead.
