**Architecture:** Three sequential phases gated on verification: (1) pnpm migration with strict isolated node_modules, (2) Magic UI component scaffolding into `components/magicui/`, (3a) landing page restructure to lead with platform/modes, (3b) chat polish with composer extraction, message-bubble extraction, streaming opacity, and border-beam accents.
