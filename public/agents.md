# Public

This directory contains static assets served by Next.js (images, SVGs, marketing assets).

For project architecture, app routing, and conventions, see:
- [AGENTS.md](../AGENTS.md) — Main project knowledge base
- [app/AGENTS.md](../app/AGENTS.md) — Next.js App Router and routes

## Key directories

- `public/images/` — Marketing and product images
- `public/marketing/` — Landing page assets
- `public/*.svg` — Icon assets

## Note

Static files here are served from the root path (`/filename`).
